# My Fancy Fences

Lekka aplikacja WPF do organizowania skrótów i folderów w panelach widocznych na pulpicie Windows.

## Funkcje

- wiele niezależnych paneli pulpitu,
- personalizacja wyglądu, nagłówków, ikon i typografii,
- automatyczne obserwowanie zawartości folderów,
- drag & drop elementów do paneli,
- przeglądarka tapet Wallhaven,
- zasobnik systemowy i opcjonalny autostart.

## Wymagania

- Windows,
- .NET 10 SDK do kompilacji projektu.

## Uruchomienie

```powershell
dotnet build "My Fancy Fences.slnx"
dotnet run --project "My Fancy Fences/My Fancy Fences.csproj"
```
